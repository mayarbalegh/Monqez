🩸 Sheryan Hayah

📝 Project Description

Sheryan Hayah is a smart application designed to organize the blood donation process by connecting blood donors with hospitals and blood banks based on real-time needs. 🚑🩸The application helps reduce response time during emergency cases, improves coordination, and contributes directly to saving lives.
